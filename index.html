许可协议

7-Zip是免费开源软件。大部分代码基于GNU LGPL许可协议，部分代码基于BSD 3-clause许可协议， 另有部分代码受unRAR许可限制。请阅读7-Zip许可协议了解更多信息。

您可以在任何计算机上使用7-Zip， 包括商业机构的计算机。无需注册或付费。

7-Zip主要特性

  • 对7z格式使用LZMA和LZMA2压缩算法可获得高压缩比
  • 支持格式：
    • 压缩/解压：7z、XZ、BZIP2、GZIP、TAR、ZIP和WIM
    • 仅解压：APFS、AR、ARJ、CAB、CHM、CPIO、CramFS、DMG、EXT、FAT、GPT、HFS、IHEX、ISO、LZH、LZMA、MBR、MSI、NSIS、NTFS、QCOW2、RAR、RPM、SquashFS、UDF、UEFI、VDI、VHD、VHDX、VMDK、XAR和Z
  • 对于ZIP和GZIP格式，7-Zip的压缩率比PKZip和WinZip高2-10%
  • 7z和ZIP格式支持强AES-256加密
  • 7z格式支持创建自解压文件
  • 与Windows Shell集成
  • 强大的文件管理器
  • 强大的命令行版本
  • FAR Manager插件支持
  • 支持87种语言本地化

7-Zip支持 Windows 11 / 10 / 8 / 7 / Vista / XP / 2022 / 2019 / 2016 / 2012 / 2008 / 2003 / 2000。
