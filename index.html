软件许可

7-Zip是开源免费软件。主要代码采用GNU LGPL许可，部分代码使用BSD 3-clause许可， 还有部分代码受unRAR许可限制。详见许可协议。

可以在任何计算机上使用7-Zip， 包括商业用途的计算机。完全免费无需注册。

主要功能

  • 对7z格式使用LZMA和LZMA2算法实现高压缩率
  • 支持格式：
    • 压缩/解压：7z、XZ、BZIP2、GZIP、TAR、ZIP和WIM
    • 仅解压：APFS、AR、ARJ、CAB、CHM、CPIO、CramFS、DMG、EXT、FAT、GPT、HFS、IHEX、ISO、LZH、LZMA、MBR、MSI、NSIS、NTFS、QCOW2、RAR、RPM、SquashFS、UDF、UEFI、VDI、VHD、VHDX、VMDK、XAR和Z
  • 对于ZIP和GZIP格式，7-Zip压缩率比PKZip和WinZip高2-10%
  • 7z和ZIP格式支持AES-256加密
  • 7z格式支持自解压文件
  • Windows Shell集成
  • 强大的文件管理
  • 命令行版本
  • FAR Manager插件
  • 支持87种语言

7-Zip支持 Windows 11 / 10 / 8 / 7 / Vista / XP / 2022 / 2019 / 2016 / 2012 / 2008 / 2003 / 2000。
